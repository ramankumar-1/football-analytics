This project aims to analyze international football match data to gain insights on team and player performance, as well as predict future match outcomes. The data used will include statistics on goals,penalties and other game-related events. Numerous data visualization techniques will be used to present the data and findings. Machine learning models will be applied to predict the outcomes of future matches based on the historical data.
